# cpp-area-calculator
This is a baby C++ project that takes the side length of a square and tells you its area 


## **How does it work?** 

-you type the side lenth in the given place

-press enter

-BAM! You got your area 🎉 


## **How to compile?** 

g++ square-calc.cpp -o square && ./square
